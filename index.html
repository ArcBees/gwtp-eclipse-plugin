GWTP Plugin For Eclipse

Eclipse 4.3 (Kepler) and below:

GWTP Plugin for Eclipse requires Eclipse 4.4 (Luna) or later.

If you have an older version of eclipse you can use the old plugin

Setup

Prerequisites

  1. Eclipse Version 4.4 (Luna) or later
  2. Google Plugin for Eclipse

Install

  1. In Eclipse select Help -> Install New Software
  2. In the dialog that appears enter the following update site:
    Update site:http://arcbees.github.io/gwtp-eclipse-plugin/4.4
  3. You may need to reset your perspective to have the GWTP wizards appear in the new shortcut menu.

Creating a Project

  1. Right Click the Package Explorer and Select New -> New GWTP Project
  2. Select the options that you want
  3. Click Finish
